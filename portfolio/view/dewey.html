DEWEY Framework (2007-2009)

What it Was

DEWEY was an extension of the YUI JavaScript library which incorporated a new standard set of reusable widgets and utilities for use across a the 60+ websites in the WDPRO portfolio. It also helped define a common set of standards and guidelines for JavaScript development across the whole organization.

What I Did

I was the lead developer for the project. I created, reviewed, and architected most of the library. Designed for ease of use, it was incorporated into many different build processes and was deployed across many diverse technology stacks and applications.

Why You Should Care

At the time I was leaving Disney, this library was the only JavaScript library approved for use on all WDPRO web sites and was gaining momentum towards being incorporated into other sites across The Walt Disney Company, including ABC.com.

As the team lead, I was responsible for evangelizing best practices and teaching developers throughout the organization how to use the library.
